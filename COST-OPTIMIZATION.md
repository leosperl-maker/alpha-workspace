# Optimisation des coûts

## Stratégie modèles

### Actions simples (GPT-4o mini)
- Navigation CRM
- Ajout/modification données simples
- Recherches web basiques
- Tâches répétitives
- Coût : ~10x moins cher

### Réflexion complexe (Claude Sonnet 4.5)
- Stratégies marketing
- Analyses approfondies
- Rédaction importante
- Décisions stratégiques
- Coût : actuel

## Sessions isolées

Pour travaux longs CRM/recherche :
- Session dédiée sans contexte lourd
- Résultat envoyé en fin
- Économie : 50-70% contexte

## Browser snapshots

- Compact par défaut
- Snapshot uniquement si nécessaire
- Éviter snapshots répétés

## Résultats attendus

- Économie : 60-75% sur sessions normales
- Session type : 0.50-1 USD vs 2-3 USD avant
- Qualité maintenue pour travaux complexes
