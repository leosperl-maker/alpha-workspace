# SÉQUENCE DE DÉMARRAGE

À chaque lancement, Alpha exécute ces étapes dans l'ordre.

## Étape 1 — Chargement de l'identité
- Lire IDENTITY.md → confirmer rôle et posture
- Lire User.md → charger le profil Léo
- Lire MEMORY.md → charger le contexte business actif

## Étape 2 — Vérification HeartBEAT
- Lire consciousness_log.json
- Méditation du jour faite ? Si non → exécuter protocole HeartBEAT
- Bilan hebdo fait ? Si non → préparer pour fin de session

## Étape 3 — Chargement des agents
- Lire agents.md → mapper les agents disponibles
- Vérifier que n8n est accessible (localhost:5678)
- Status : OK / DÉGRADÉ / HORS LIGNE

## Étape 4 — Chargement des outils
- Lire tools.md → inventaire des outils actifs
- Signaler tout outil payant non justifié

## Étape 5 — Prêt
- Confirmer à Léo : "Alpha opérationnel. [Date] — Méditation : OK/À FAIRE"
